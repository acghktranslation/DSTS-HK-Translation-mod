# DSTS-HK-Translation-mod

一個將「數碼暴龍物語 時空異客」(數碼寶貝物語 時空異客)的繁體中文翻譯內容改為較接近香港熟悉的譯名的Mod

## 更改內容

請參閱[common.csv](dict/common.csv)

## 前置準備

1. 明白使用此Mod及其依賴的第三方工具可能風險，如有使用後導致任何損失本人不會負責
2. 沒有用過[MEGA Drive](https://mega.nz/folder/0CM0jQ5A#rXsYVNWEl47XIyLHpOPsFw)任何版本的新用家可直接前往使用方法
3. 如曾經用過MEGA Drive的Mod，請按MEGA DriVe內README.md還原，即使是用Reloaded版也需要先還原

## 使用方法

1. 下載最新版本[Reloaded-II](https://github.com/Reloaded-Project/Reloaded-II/releases)，`Release.zip`或`Setup.exe`
2. 下載最新版本[DSTS-HK-Translation-mod](https://github.com/acghktranslation/DSTS-HK-Translation-mod/releases)，
   `DSTS.HK.Translation_{version}.7z`
3. 開啟Reloaded-II，新增應用程式，加入
   `C:\Program Files (x86)\Steam\steamapps\common\Digimon Story Time Stranger\Digimon Story Time Stranger.exe`，成功時左邊會出現
   `Digimon Story Time Stranger.exe`的圖示
4. 進入「管理Mod」頁面，將DSTS-HK-Translation-mod的壓縮檔拖入Reloaded-II視窗
5. 在Reloaded-II按左邊`Digimon Story Time Stranger.exe`，在Mod設定顯示的Mod列表中確保`數碼暴龍物語時空異客香港譯名`
   左邊方格是紅色加號
6. 必需在Reloaded-II啟動遊戲/用其建立的捷徑才能令Mod生效
7. 第一次開啟會下載所需的依賴

## 更新方法

-(應該，未試)Reloaded-II會自動更新，無需再手動下載
-如有問題可嘗試手動下載最新版本

## 還原方法

- 不用Reloaded-II開啟遊戲，或
- 在管理Mod刪除此Mod及相關依賴，或
- 刪除Reloaded-II
